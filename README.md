i-cant-afford-italian

A broke person's date funding page. Track upcoming dates, see how much they cost, and let people chip in so I don't go bankrupt keeping this one.

Why this exists

I don't pull shatty baes. I pull good ones. I just can't afford Italian food on my own, so this project closes that gap.

What it does


Track dates: log where we're going, when, and what it costs
Show funding progress: see how much of the date is covered vs still needed
Let people contribute: you guys can pitch in toward a specific date
Running tally: total spent, total raised, current bank account status


Status

Early build.

Roadmap


Basic UI to list dates and costs
Contribution and payment tracking
Progress bar per date
Total spent vs total raised summary
Optional leaderboard for top contributors


Tech stack


HTML
React
CSS


Contributing

This relationship depends on you guys. If you want to fund a date, open the app, pick a date, send help. If you want to contribute code instead of money, PRs welcome, it's the same energy.
